# 📡 内容雷达 Content Radar

> 帮知识博主找到**"有信息差 + 有受众"**的选题。一条命令，全网扫描。

## 四个工具

| 工具 | 触发词 | 功能 |
|------|--------|------|
| 📡 **内容雷达** | "内容雷达"、"找选题"、"今天发什么" | 全网扫描，推荐 5-8 个选题 |
| 🔍 **爆款拆解** | "拆解 [URL]" | 深度拆解单条爆款内容 |
| 📦 **批量拆解** | "批量拆解 URL1 URL2 ..." | 多条内容横向对比，找差异化空间 |
| 💎 **素材提取** | "提取素材 [URL]" | 提取金句/数据/案例，跳过分析 |

先用「内容雷达」找到值得做的选题，再用「爆款拆解」深入学习爆款的创作技巧。

## 它能做什么

### 📡 内容雷达

输入"内容雷达"（或"找选题"、"今天发什么"），AI 自动帮你：

1. **自动检测工具状态** → 缺什么自动装，装不了的明确告知
2. **扫描海外前沿**（Twitter、YouTube、GitHub、Exa、RSS）→ 发现别人还没讲的新东西
3. **扫描国内热帖**（小红书、B站、微信公众号、抖音）→ 找到受众正在关心的话题
4. **交叉比对**（供给 × 需求）→ 推荐 5-8 个选题，按信息差和受众匹配度排序
5. **附带证据链**（热帖链接 + 学习材料 URL + 信源质量标签 + 时效标签）→ 每个推荐都有数据支撑

### 🔍 爆款拆解

输入"拆解 [URL]"，AI 自动帮你：

1. **提取全文内容**（视频字幕 / 文章正文 / 帖子评论）
2. **四维深度分析**：
   - 🏗️ 内容结构拆解 — 开头 Hook、叙事节奏、CTA 设计
   - 💬 受众反馈分析 — 评论情绪、高频提问、未满足需求
   - 🎯 标题与包装技巧 — 标题公式、标签策略、平台适配
   - 🎪 选题跟进建议 — 值不值得做、差异化角度、标题方案
3. **输出完整报告**（聊天速览 + 文件保存）

### 📦 批量拆解

输入"批量拆解 URL1 URL2 URL3"，AI 帮你横向对比多条内容：

- 生成**角度矩阵**（🔴红海 / 🟡浅提 / 🟢蓝海）
- 找到**差异化空间**
- 汇总**高频素材**

### 💎 素材提取

输入"提取素材 [URL]"，快速提取可复用素材：

- 可引用金句（带时间戳）
- 关键数据点（带出处）
- 案例/故事片段
- 评论区高频问题

## 30 秒安装

### 第一步：下载

```bash
git clone https://github.com/iamgxd01-lgtm/content-radar.git
cd content-radar
```

### 第二步：一键安装

```bash
bash setup.sh
```

安装脚本会自动帮你装好所有需要的工具。如果某个工具没装成功也不用担心——内容雷达会**在运行时自动检测并补装缺失的工具**。

### 第三步：开始使用

打开你的 AI 编辑器，输入：

```
内容雷达
```

或者拆解一条爆款内容：

```
拆解 https://www.youtube.com/watch?v=xxx
```

首次运行会问你 3 个问题（你的领域、发布平台、创作风格），之后就是全自动的。

> **没运行 setup.sh 也没关系**——第一次输入"内容雷达"时，AI 会自动检测并安装缺失的工具。

## 配置文件

首次运行后，你的配置会保存在 `~/.content-radar/my-radar.yaml`。

你可以随时编辑这个文件来调整：
- 搜索关键词（想搜什么就搜什么）
- 发布平台（决定扫描哪些国内平台）
- 评分权重（更看重时效性还是信息差？维度名也可以自定义！）
- 关注的信息源（Twitter 账号、RSS 订阅）
- 能力圈过滤（只推荐你能做的选题类型）
- 选题分类（给选题打标签，方便安排内容日历）

配置示例见 `examples/my-radar.yaml.example`。

## 采集渠道

| 渠道 | 工具 | 用途 |
|------|------|------|
| Twitter/X | xreach | 一手讨论，比博客早 3-7 天 |
| YouTube | yt-dlp | 视频教程和深度讨论 |
| GitHub | gh CLI | 开源项目和代码动态 |
| Exa | mcporter | 语义搜索，精准查找 |
| RSS | feedparser | 官方博客订阅 |
| 微信公众号 | miku-ai | 国内深度文章 |
| 小红书 | mcporter (MCP) | 国内热帖和受众需求 |
| B站 | yt-dlp | 国内视频竞品 |
| 抖音 | agent-reach | 短视频爆款和热度 |

每个渠道都有自动降级策略——如果某个工具不可用，会自动切换到备用方案，并在输出中标注信源质量（🟢一手 / 🟡二手 / 🔴降级）。

## 常见问题

### 选题太泛怎么办？

内容雷达会优先推荐 24 小时内的热点内容。如果需要更强的时效性，可以在配置中提高"时效性"的权重：

```yaml
scoring:
  时效性: 40    # 提高到 40%
  信息差: 30
  受众匹配: 20
  可操作性: 10
```

### 工具不可用怎么办？

内容雷达会在启动时**自动检测并安装**缺失工具（pip3/npm 包）。只有需要手动认证的工具会提示你操作：

- Twitter 数据需要认证：`xreach auth`
- GitHub 数据需要认证：`gh auth login`

其余工具都是自动安装、自动降级，无需手动处理。

更多问题见 [docs/TROUBLESHOOTING.md](docs/TROUBLESHOOTING.md)
