# 📡 内容雷达 Content Radar

> 帮知识博主找到**"有信息差 + 有受众"**的选题。一条命令，全网扫描。

## 两个工具

| 工具 | 触发词 | 功能 |
|------|--------|------|
| 📡 **内容雷达** | "内容雷达" | 全网扫描，推荐 5-8 个选题 |
| 🔍 **爆款拆解** | "拆解 [URL]" | 深度拆解单条爆款内容 |

先用「内容雷达」找到值得做的选题，再用「爆款拆解」深入学习爆款的创作技巧。

## 它能做什么

### 📡 内容雷达

输入"内容雷达"，AI 自动帮你：

1. **扫描海外前沿**（Twitter、YouTube、GitHub、Exa、RSS）→ 发现别人还没讲的新东西
2. **扫描国内热帖**（小红书、B站、微信公众号）→ 找到受众正在关心的话题
3. **交叉比对**（供给 × 需求）→ 推荐 5-8 个选题，按信息差和受众匹配度排序
4. **附带证据链**（热帖链接 + 学习材料 URL）→ 每个推荐都有数据支撑

### 🔍 爆款拆解

输入"拆解 [URL]"，AI 自动帮你：

1. **提取全文内容**（视频字幕 / 文章正文 / 帖子评论）
2. **四维深度分析**：
   - 🏗️ 内容结构拆解 — 开头 Hook、叙事节奏、CTA 设计
   - 💬 受众反馈分析 — 评论情绪、高频提问、未满足需求
   - 🎯 标题与包装技巧 — 标题公式、标签策略、平台适配
   - 🎪 选题跟进建议 — 值不值得做、差异化角度、标题方案
3. **输出完整报告**（聊天速览 + 文件保存）

## 30 秒安装

### 第一步：下载

```bash
git clone https://github.com/iamgxd01-lgtm/content-radar.git
cd content-radar
```

### 第二步：一键安装

```bash
bash setup.sh
```

安装脚本会自动帮你装好所有需要的工具。如果某个工具没装成功也不用担心——内容雷达会自动用其他方式替代。

### 第三步：开始使用

打开你的 AI 编辑器，输入：

```
内容雷达
```

或者拆解一条爆款内容：

```
拆解 https://www.youtube.com/watch?v=xxx
```

首次运行会问你 3 个问题（你的领域、发布平台、创作风格），之后就是全自动的。

## 配置文件

首次运行后，你的配置会保存在 `~/.content-radar/my-radar.yaml`。

你可以随时编辑这个文件来调整：
- 搜索关键词（想搜什么就搜什么）
- 发布平台（决定扫描哪些国内平台）
- 评分权重（更看重时效性还是信息差？）
- 关注的信息源（Twitter 账号、RSS 订阅）

配置示例见 `examples/my-radar.yaml.example`。

## 采集渠道

| 渠道 | 工具 | 用途 |
|------|------|------|
| Twitter/X | xreach | 一手讨论，比博客早 3-7 天 |
| YouTube | yt-dlp | 视频教程和深度讨论 |
| GitHub | gh CLI | 开源项目和代码动态 |
| Exa | mcporter | 语义搜索，精准查找 |
| RSS | feedparser | 官方博客订阅 |
| 微信公众号 | miku-ai | 国内深度文章 |
| 小红书 | mcporter | 国内热帖和受众需求 |
| B站 | yt-dlp | 国内视频竞品 |

每个渠道都有自动降级策略——如果某个工具不可用，会自动切换到备用方案。
